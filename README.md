# Cricket Application Form

A web-based application form built using HTML and CSS for collecting player details for cricket registration.

## Features

- Clean and responsive form UI
- Collects user details such as name, age, contact, and role
- Image upload support
- Simple and user-friendly layout

## Tech Stack

- HTML
- CSS

## Functionality

This form is connected to a backend system using Google Apps Script, which stores:

- Form data → Google Sheets (as database)
- Uploaded images → Google Drive

## Related Project

This project is integrated with:
Cricket_Application_Sheet (database handling system)

## Purpose

The project demonstrates:
- Frontend form design
- Integration with cloud-based storage
- Real-time data handling using Google services
